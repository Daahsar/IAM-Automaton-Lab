\# Phase 2 — Access Control, Governance \& Monitoring



Phase 2 expands the IAM Automation Lab beyond identity provisioning by introducing automated

access control, governance workflows, and monitoring capabilities. This phase builds on the

identity baseline established in Phase 1 and begins shaping the lab into a full IAM automation

framework.



\## Core Components



\### 1. RBAC Automation

\- JSON-based role definitions

\- Automated role assignment via Microsoft Graph

\- Least-privilege access patterns

\- Group-based role assignment support



\### 2. Conditional Access Automation

\- JSON-based CA policy templates

\- Automated deployment via Microsoft Graph

\- Baseline security policies (MFA, compliant device, location-based)

\- Environment-specific policy bundles



\### 3. Identity Governance

\- Automated access review creation and scheduling

\- Entitlement management structure (catalogs, access packages)

\- Governance lifecycle documentation

\- Integration with monitoring outputs



\### 4. Monitoring \& Audit Extraction

\- Sign-in log extraction

\- Audit log extraction

\- JSON/CSV export for reporting

\- Foundation for future dashboards and visualization



\## Deliverables

\- /Phase2/RBAC/ — role templates + automation scripts

\- /Phase2/Governance/ — access review automation
